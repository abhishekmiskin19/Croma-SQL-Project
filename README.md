# 📊 Croma Electronics Inventory Analysis using SQL

## 📌 Project Overview

This project focuses on analyzing an e-commerce product inventory database using SQL. The dataset simulates a real-world scenario of an electronics store (Croma), where products and their manufacturers are stored in relational tables.

The goal of this project is to perform data extraction, filtering, aggregation, and relational analysis using SQL queries.

---

## 🗂️ Database Structure

### 1️⃣ Product_Inventory Table

* Product_Code (Primary Key)
* Product_Name
* Product_Price
* Manufacturer_Code (Foreign Key)

### 2️⃣ Manufacturers Table

* Manufacturer_Code (Primary Key)
* Manufacturer_Name

---

## 🔗 Relationship

* One Manufacturer → Many Products
* Connected via `Manufacturer_Code`
